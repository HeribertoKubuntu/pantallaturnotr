# Turnos en Pantalla Salud Digna

## ¿Qué es?
Es una aplicación para que nuestros pacientes puedan observar de manera visual el orden de atención en sucursal.

<hr>

## Objetivo
Ofrecer un mejor servico a nuestros pacientes.
